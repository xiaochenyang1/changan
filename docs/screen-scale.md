# 大屏自适应方案 - Scale 缩放

## 实现方式

使用 **scale 缩放** 方案，自动将设计稿尺寸（1920x1080）等比例缩放填充整个屏幕。

## 配置

| 设计稿尺寸 | 1920 x 1080 |
|-----------|-------------|
| 缩放方式 | 按屏幕比例自动缩放 |
| 居中显示 | 是 |

## 使用方式

### 1. 页面样式编写

直接使用设计稿的 px 尺寸编写样式，无需手动转换：

```vue
<style scoped>
.box {
  width: 1920px;    /* 直接写设计稿尺寸 */
  height: 1080px;
  padding: 40px;
  font-size: 24px;
}
</style>
```

### 2. 工作原理

```
屏幕尺寸 1920x1080 → 缩放比例 1 → 页面满屏显示
屏幕尺寸 1280x720  → 缩放比例 0.667 → 页面等比例缩小
屏幕尺寸 2560x1440 → 缩放比例 1.33 → 页面放大
```

## 优点

1. **宽高同时缩放** - 不管屏幕比例如何，内容始终等比例显示
2. **字体不变形** - 使用 scale 缩放，所有元素按统一比例缩放
3. **简单易用** - 直接使用设计稿 px 值，无需计算转换
4. **自动居中** - 屏幕比例大于设计稿时自动水平垂直居中
